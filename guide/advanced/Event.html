    # 事件监听

    ProForm 支持监听内部组件事件。

    默认监听事件名格式为 field:eventName

    # 方式

    // 假设我们需要监听"姓名"输入框的input事件,写法如下
<pro-form
  @personname:input="inputEvent">
</pro-form>
